1. What is a module?
	- Một hay nhiều function, có thể export, import khi cần thiết
2. What is package?
	- Một module hoặc tập hợp các module
3. What is NPM?
	- Node Package Manager
	- trình quản lý package mặc định cho Node.js, cho phép người dùng sử dụng và phân phối các module, tạo, sử dụng, phân phối package
4. What is Webpack?
	- là công cụ giúp người dùng compile các module Javascript
5. How do you create a new React project?
	- cài đặt node
	- chạy lệnh: npx create-react-app <name of project>
6. What are the files and folders inside a project folder(package.json, package-lock.json or yarn.lock, .gitignore,node_modules and public)?
	- package.json: liệt kê các package mà app dùng
	- package-lock.json or yarn.lock: a means to lock the version of the package
	- .gitignore: tạo ra khi khởi tạo git, cho phép các file và folder không bị đẩy lên git remote
	- node_modules: lưu trữ tất cả node package cần thiết cho app
	- public
